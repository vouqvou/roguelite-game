# Core Gameplay Loop

## Second-to-second
- Move (WASD), dodge, attack. Manage positioning vs swarms.

## Wave (30-60s)
1. Enemies spawn from edges in scripted composition.
2. Player clears all enemies.
3. Reward screen: choose 1 of 3 randomized skills.
4. Short breather, then next wave (higher difficulty).

## Run (15-25 min)
- 20 waves, boss every 5th (waves 5, 10, 15, 20).
- Death ends run; meta shards granted based on waves reached.

## Meta (between runs)
- Spend shards on permanent unlocks (new skills, base stats, characters).
