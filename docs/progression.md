# Progression

## In-Run (temporary)
- One skill chosen per wave cleared (~19 picks/full run).
- Skills stack; rarity weights shift toward rare/epic in later waves.
- Skill categories: Offense, Defense, Utility, Synergy.

## Meta (permanent)
- Currency: Shards (earned per wave reached + boss kills).
- Unlock tree:
  - Base HP / damage / move speed tiers.
  - New starting characters (different base skill).
  - New skills added to the run reward pool.
  - Quality-of-life (extra reroll, banish).

## Reward pacing (shards)
- +2 per wave cleared, +10 per boss, +25 first-clear bonus.
